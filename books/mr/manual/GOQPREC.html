GOQPREC

GOQPREC, Numeric precision in key error: Blk #xxxx, Key #yyyy. Record not loaded.

MUPIP Information: This indicates that GT.M was unable to precisely represent a key in the GOQ input file to a MUPIP LOAD.

Action: Examine the key on the source system, modify it, and repeat the process, or manually enter the modified record into GT.M.
