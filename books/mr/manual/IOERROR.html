IOERROR

IOERROR, Error occurred while doing aaaa in oooo operation -- called from module mmmm at line LLLL

Run Time Error: On UNIX this indicates a system call used to manage the underlying O/S device for a FIFO, PIPE or Sequential Disk failed in a way that GT.M did not anticipate.

Action: Use the OS documentation to investigate the failure.
