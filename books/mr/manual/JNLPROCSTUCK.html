JNLPROCSTUCK

JNLPROCSTUCK, Journal file writes blocked by process xxxx

Run Time Warning: This indicates that a GT.M process waited for nearly one minute and is not able to flush the journal as the journal write mechanism seems to be blocked by process xxxx.

Action: If the situation does not improve, kill the offending process xxxx. This may indicate an overloaded disk subsystem on which journal file xxxx resides. Consider balancing disk subsystem load. If necessary, report the entire incident context with operator log information to your GT.M support channel.
