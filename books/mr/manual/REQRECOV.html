REQRECOV 

REQRECOV, Error accessing database dddd. Must be recovered on cluster node ccccc.

Run Time Error: This indicates that GT.M could not open a previously journaled database file dddd due to a prior improper shutdown on cluster node ccccc. A GT.M process on cluster node ccccc may have failed to attach a database memory segment or it was terminated by a method other than MUPIP STOP.

Action: Perform a MUPIP JOURNAL RECOVER operation to address this issue.
