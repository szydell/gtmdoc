BTFAIL 

BTFAIL, The database block table is corrupt; error type xxxx

Run Time Error: This indicates that a database operation failed because the tables of recently used blocks are damaged. GT.M uses the block tables to control and optimize database traffic.

Action: Report this database cache error to the group responsible for database integrity at your operation.
