BREAKZBA

BREAKZBA, Break instruction encountered during ZBREAK action

Run Time Information: This indicates that GT.M encountered a BREAK command within a ZBREAK action string and entered Direct Mode. The ZBREAK command sets or clears breakpoints during debugging. All commands entered at the Direct Mode prompt are compiled and executed as they are entered.

Action: To continue program execution, enter the ZCONTINUE command.
