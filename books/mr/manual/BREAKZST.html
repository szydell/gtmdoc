BREAKZST

BREAKZST, Break instruction encountered during ZSTEP action

Run Time Information: This indicates that GT.M encountered a BREAK command within a ZSTEP action string and entered Direct Mode.

Action: The ZSTEP command causes GT.M to proceed to the beginning of the next line of M code that matches the characteristic specified by the ZSTEP argument. GT.M compiles and executes all commands entered at the Direct Mode prompt as they are entered. To continue program execution, enter the ZCONTINUE command.
