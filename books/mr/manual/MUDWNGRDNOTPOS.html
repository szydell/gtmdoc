MUDWNGRDNOTPOS

MUDWNGRDNOTPOS, Start VBN value is [xxx] while downgraded GT.M version can support only [yyy]. Downgrade not possible

MUPIP Error: Older versions of GT.M require the first GDS block be at Virtual Block Number yyy but it is at VBN xxx. This is likely due to the file initially being created using a newer version of GT.M and thus cannot be downgraded.

Action: To use the database with an older version of GT.M, it must be extracted with the current version and loaded into the older version both in ZWR format.
