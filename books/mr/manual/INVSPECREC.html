INVSPECREC 

INVSPECREC, Invalid global modifier record

Run Time Error: This indicates that GT.M could not access the header information for a global; therefore, it could not determine the collation characteristics of the global.

Action: Use the %gbldef utility routine to investigate the current state of the global header and correct it.
