SEMUNDOOVERFLOW

SEMUNDOOVERFLOW, The operating system per-process SEM_UNDO table is full and region rrrr cannot be opened

Run Time Error: The number of database files in use exceeds the per-process operating system limit for handling SEM_UNDO semaphore operations.

Action: Reduce the number of database regions processed at one time.
