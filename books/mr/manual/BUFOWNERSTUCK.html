BUFOWNERSTUCK

BUFOWNERSTUCK, PID xxxx waiting for PID yyyy to finish disk read of block zzzz. Been waiting for aaaa minutes.

Run Time Warning: Poor response time from the I/O subsystem could cause this error, or if the process performing the disk-read was suspended.

Action: If the disk-reading process is found to be in a process-suspended state, un-suspend it. If it is not suspended, examine the I/O subsystem performance characteristics for behavior problems. Report the full error message, along with the GT.M operator log messages during the specific timeframe, to your GT.M support channel.
