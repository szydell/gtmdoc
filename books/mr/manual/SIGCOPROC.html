SIGCOPROC

SIGCOPROC, Signal was caused by a coprocessor error

Run Time Error: This message is an auxiliary message and is preceeded with a primary KILLBYSIGxxx message.

Action: Refer to the accompanying message(s) and take appropriate action. Refer to the UNIX user documentation. If necessary, report the entire incident context to your GT.M support channel for further analysis.
