DBRNDWNWRN

DBRNDWNWRN, Global database xxxx not rundown successfully by PID yyyy [zzzz]. Global section was not removed.

Run Time Error: When the last process attached to a GT.M database shared memory segment or global section detaches from the same, it normally removes the segment/section from the system. In case of an error while flushing the contents from the segment/section to the database file on disk, this removal is not done and this error is issued.

Action: Attempt a MUPIP RUNDOWN on that region. In case of an error, attempt corrective action corresponding to the displayed error.
