AUDLOGFAIL

AUDLOGFAIL, Audit XXXX facility failed to log activity

Run Time Error: GT.M was unable to send the to-be-logged activitiy by XXXX to logger. This prevents a process from taking the action when using XXXX.

Action: Check to make sure that GT.M is able to successfully connect to the logger program. Check syslog or additional messages for more information.
