NOCREMMBIJ

NOCREMMBIJ, MM access method not compatible with BEFORE image journaling; Database file DDDD not created

MUPIP/Run Time Error: While creating a database file with access mode MM, GTM found the database file DDDD to be configured for BEFORE image journaling which is not supported with that access method.

Action: Either change the access mode of the file or do not enable BEFORE image journaling and retry.
