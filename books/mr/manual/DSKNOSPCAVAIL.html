DSKNOSPCAVAIL

DSKNOSPCAVAIL, Attempted write to file FFFF failed due to lack of disk space. Retrying indefinitely.

Run Time Error: This error indicates that GT.M could not update file FFFF due to lack of disk space in the file system. If -INST_FREEZE_ON_ERROR is enabled, GT.M automatically disables it (sending a DSKSPCAVAILABLE message to the operator log) when adequate disk space becomes available again.

Action: Make disk space available in the file system to allow updates to file FFFF.
