CRITSEMFAIL

CRITSEMFAIL, Error with semaphores for region xxxx

Run Time Error: This indicates that GT.M encountered a missing or damaged semaphore. This typically indicates that an agent external to GT.M has deleted or modified the semaphores GT.M uses to manage database and LOCK interactions.

Action: Investigate the state of the GT.M semaphores and all previous actions that might have damaged them.
