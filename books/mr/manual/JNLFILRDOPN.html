JNLFILRDOPN

JNLFILRDOPN, Error opening journal file xxxx for read for database file yyyy

Source Server log/MUPIP Error: This indicates that GT.M was unable to open journal file xxxx in read-only mode for the specified database file. The Source Server exits with a JNLFILRDOPN message after six failed attempts to open journal files.

Action: Review the accompanying message(s) for additional information.
