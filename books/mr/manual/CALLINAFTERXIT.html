CALLINAFTERXIT

CALLINAFTERXIT, After a gtm_exit, a process can never create a valid GT.M context

Run Time Error: Once a call-in has done a call to gtm_exit(), a process can no longer do GT.M call-ins

Action: Either move or remove the inappropriate call-ins or move the gtm_exit call to a later point.
