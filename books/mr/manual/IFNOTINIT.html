IFNOTINIT

IFNOTINIT, External Interface must first call GTM$INIT or M routine

Run Time Error: This indicates that an external routine could not access a GT.M database library routine because it did not call the GTM$INIT library routine first. The GTM$INIT library routine initializes the GT.M run-time environment.

Action: Call GTM$INIT or a GT.M M routine before calling any other database access library routine.
