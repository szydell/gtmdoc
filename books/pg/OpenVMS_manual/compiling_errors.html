Compiling Errors

When there are errors, external call table creation generates an error summary with line numbers. Examples of types of reported errors include:

  • A required keyword has not been specified.

  • Keyword is not valid

  • A specified mechanism does not exist.

  • INPUT= and OUTPUT= specifications in the ROUTINE line do not match the actual number of INPUT and OUTPUT lines.

  • INPUT lines do not precede the OUTPUT lines.
