/SH[AREABLE]

Informs the Linker that the specified file is a shareable image with a default file extension .EXE. This qualifier is only valid in an options file. When you use this qualifier as an argument qualifier on a LINK command line, the Linker interprets it as a command qualifier. For an introduction to shareable images, see the above description of the /SHAREABLE command qualifier.
