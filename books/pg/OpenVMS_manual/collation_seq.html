Establishing A Local Collation Sequence

Normally, GT.M orders data with numeric values first, followed by strings sequenced by ASCII values, as is required by the M standard. To use an alternative collating sequence the following items must be provided prior to invocation of GT.M.

  • The logical name GTM_COLLATE_n, specifying the executable image containing the alternative sequencing routines

  • The executable image
