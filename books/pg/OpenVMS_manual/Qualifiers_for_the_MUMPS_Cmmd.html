Qualifiers for the MUMPS Command

The MUMPS command allows qualifiers that customize the type and form of the compiler output to meet specific programming needs. MUMPS command qualifiers may also appear as a modifier to the argument to the GT.M ZLINK and ZCOMPILE commands. The following section describes the MUMPS command qualifiers.
