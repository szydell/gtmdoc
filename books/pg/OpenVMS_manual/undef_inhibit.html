UNDEF_INHIBIT

Specifies a truth value indicating whether GT.M treats undefined variables as $GET() functions, that is, as null strings.

This factor can be modified at run-time by using the VIEW command with an argument having "UNDEF" or "NOUNDEF" for a value.
