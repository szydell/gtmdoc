Input Arguments

The input arguments for gtm_ac_xform are:

argument 1: a string subscript, passed by static descriptor that contains the length of the string.

argument 2: an unsigned integer which gives the subscript level; this is not used currently, but is reserved for future enhancements.

argument 3: a string buffer to be filled with the transformed key.
