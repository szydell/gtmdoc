Image

IM[AGE]=strlit

The string literal specifies the file-specification of the image for the resultant job to activate.

GT.M does not supply a default file type. Image files generally have a .EXE extension. A JOB command requires a GT.M image that has been linked under a compatible version of GT.M.

The maximum string length is 255 characters.

By default, JOB provides the executing current image.
