Examples of New

Example:

        SET A(1)=1,B=4,C=5
        WRITE !,"VARIABLES BEFORE NEW:",!
        ZWRITE
        DO LABEL
        WRITE !,"VARIABLES AFTER RETURN:",!
        ZWRITE
        QUIT
        LABELNEW A SET C=7
        WRITE !,"VARIABLES AFTER NEW:",!
        ZWRITE
        QUIT

Produces the results:

        VARIABLES BEFORE NEW:
        A(1)=1
        B=4
        C=5
        VARIABLES AFTER NEW:
        B=4
        C=7
        VARIABLES AFTER RETURN:
        A(1)=1
        B=4
        C=7

Example:

        SET (A,B,C,D)="TEST"
        DO LABEL
        WRITE !,"VARIABLES AFTER RETURN:",!
        ZWRITE
        QUIT
        LABELNEW (B,C) SET (A,B,Z)="NEW"
        WRITE !,"VARIABLES AFTER EXCLUSIVE NEW:",!
        ZWRITE
        QUIT

Produces the results:

        VARIABLES AFTER EXCLUSIVE NEW:
        A="NEW"
        B="NEW"
        C="TEST"
        Z="NEW"
        VARIABLES AFTER RETURN:
        A="TEST"
        B="NEW"
        C="TEST"
        D="TEST"
