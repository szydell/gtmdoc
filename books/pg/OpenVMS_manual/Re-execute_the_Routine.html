Re-executing the Routine

Re-display the DO command using the RECALL command.

Execute the routine using the DO command.

Example:

          GTM>rec d
        GTM>D ^dmex
        
        Name: Revere, Paul
        Paul Revere
        Name: q

This example illustrates a successful execution of dmex.
