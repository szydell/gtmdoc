Output

OUT[PUT]=strlit

The string literal specifies a value for the logical name SYS$OUTPUT.

The maximum string length is 255 characters.

By default, JOB constructs the output file-specification from the ROUTINENAME using a file extension of .MJO and the current default directory of the process created by the JOB command.
