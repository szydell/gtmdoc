File Argument

This argument to the GTM$SETGBLDIR() function is a VMS descriptor that specifies the file name that GT.M should henceforth treat as the current global directory. The GTM$SETGBLDIR(file) function translates to the M statement "set $zgbldir=file". Refer to the "Intrinsic Special Variable" chapter in this manual for more detail on $ZGBLDIR.

Any error while accessing/opening the specified file causes an error status return.
