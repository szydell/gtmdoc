GTM$DEFAULTS.M64

The format of GTM$DEFAULTS.M64 is:

        .TITLE GTM$DEFAULTS
        GTM$USER_STACK_SIZE == 16384
        GTM$USER_INDRCACHE_SIZE == 32
        GTM$USER_STRPL_SIZE == 20480
        GTM$USER_IO_TIMER == 3300000
        GTM$USER_WRITE_FILTER == 0
        GTM$SPECIAL_INPUT == ^X000000
        GTM$UNDEF_INHIBIT == 0
        GTM$CTRLC_ENABLE == 1
        GTM$BREAK_MESSAGE_MASK == 15
        GTM$LOWER_CASE_LABELS == 1
        GTM$LVNULLSUBS == 1
        GTM$ZDIR_FORM == 0
        GTM$ZDATE_FORM == 0
        .END
