STARTUP

STA[RTUP]=strlit

The string literal specifies a command file that the resultant job executes before executing the GT.M routine. The command line may invoke a DCL script or other images.

If the file-specification for the startup file does not include a directory name, GT.M searches for the file in the current default directory. GT.M does not supply a default file type. If the JOB command does not specify a startup file, VMS executes only the "normal" command procedures.
