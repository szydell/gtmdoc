The GT.M Compiler

The GT.M compiler operates on source files to produce object files consisting of position-independent, shareable, native object VMS code native object code, which on some platforms can be linked into shared object libraries. GT.M provides syntax error checking at compile-time and allows you to enable or disable the compile-as-written mode. By default, GT.M produces an object file even if the compiler detects errors in the source code. This compile-as-written mode facilitates a flexible approach to debugging.
