ZDIR_FORM

"ZDIR_FORM":"value"

Includes (value=0) or excludes (value=1) the device specification in the special variable $ZDIRECTORY. By default, GT.M includes the device specification in $ZDIRECTORY. Refer to the "Intrinsic Special Variables" chapter in this manual for more information.

Note that the value of ZDIR_FORM does not affect the value to which $ZDIRECTORY is SET. ZDIR_FORM only modifies the way $ZDIRECTORY is presented. Regardless of the value of ZDIR_FORM, $ZDIRECTORY can be SET to a path that may or may not include device specification.

Linking a GT.M image with a modified GTM$DEFAULTS alters the default for that image. For more information on GTM$DEFAULTS see the "Program Development Cycle" chapter in this manual.
