Error

ERR[OR]=strlit

The string literal specifies a value for the logical name SYS$ERROR.

The maximum string length is 255 characters.

By default, JOB constructs the error file-specification from the ROUTINENAME using a file extension of .MJE and the current directory of the process created by the JOB command.
