Creating and Editing a Source Program

Editing from GT.M
Editing from DCL

The first step in developing a GT.M program is to create a source file. In most cases, the user can create and modify GT.M source programs using VMS text editors. You can create and edit a source program from either GT.M or DCL.

When the program is very simple (and its lines do not need revision after they are entered), you can use the DCL command CREATE and redirect the output to your M source file. For more information on CREATE, refer to the VMS DCL Dictionary.
