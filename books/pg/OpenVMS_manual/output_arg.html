Output Arguments

The output arguments for gtm_ac_xform are:

return value: a longword result providing a status conforming to the calling standard; it indicates the success or failure of the transformation.

argument 3: a transformed subscript in the string buffer, passed by the descriptor.

argument 4: an unsigned longword, passed by reference, giving the actual length of the output string.
