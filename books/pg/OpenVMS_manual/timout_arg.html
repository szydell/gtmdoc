Timeout Argument

The timeout specifies the time in seconds to wait before GT.M drops the attempt to LOCK or ZALLOCATE a resource name. The operation waits for the LOCK indefinitely when the timeout value specifies a negative number. When the timeout is zero (0), GT.M makes exactly one attempt to LOCK or ZALLOCATE the resource name.
