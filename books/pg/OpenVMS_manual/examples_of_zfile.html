Examples of $ZFILEATTRIBUTES()

Example 1:

    GTM> WRITE $ZFILEATTRIBUTES("GTMSHR","PRO")
    RWED,RWED,RE,RE
    GTM>

This uses $ZFILEATTRIBUTES() to display the UIC protection mask for GTMSHR.EXE.

Example 2:

    GTM> WRITE $ZFILE("M.DAT","deq")
    100
    GTM>

This uses $ZFILEATTRIBUTES() to examine the default extension size for M.DAT.
