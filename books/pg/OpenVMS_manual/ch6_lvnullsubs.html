LVNULLSUBS

"[NO]LVNULLSUBS"

Allows or disallows local arrays to have null subscripts. The default is LVNULLSUBS. Linking a GT.M image with a modified GTM$DEFAULTS alters the default for that image.For more information on GTM$DEFAULTS see the "Program Development Cycle" chapter in this manual.
