LVNULLSUBS

Specifies a truth value indicating whether GT.M permits null subscripts in local arrays or not.

This factor can be modified at run-time by using the VIEW command with an argument having "LVNULLSUBS" or "NOLVNULLSUBS" for a value.

The exact spelling of parameter names is important. Misspelled names may cause Linker warnings, and prevent changes to GTM$DEFAULTS from having the desired effect.
