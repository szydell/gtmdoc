/S[PACE]=lines

Controls the spacing of the output in the listing file. /SPACE=n specifies n-1 blank lines separating every source line in the listing file. If n<1, the MUMPS command uses single spacing in the listing.

If this qualifier appears without the /LIST qualifier, the GT.M compiler ignores the /SPACE qualifier.

By default, listings use single spaced output (/SPACE=1).
