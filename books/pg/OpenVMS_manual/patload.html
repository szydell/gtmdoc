PATLOAD

"PATLOAD":"file-specification"

Identifies the file containing definitions of unique patterns for use with the "?" operator. These pattern definitions can be used in place of, or in addition to, the standard C, N, U, L, and P. For additional information on creating the file-specification see the "Internationalization" chapter of this manual.
