ZDATE_FORM

Determines whether four digit year code is active for $ZDATE() function. GT.M defaults to zero (0), that is, two digit output. Refer to the "$ZDATE" section in the "Functions" chapter in this manual for its usage.

If no value is given with the VIEW command, it turns four digit code on. It is equivalent to the intrinsic special variable $ZDATEFORM. Use $ZDATEFORM to set this VIEW keyword. Also, logical name gtm_zdate_form may be used to set the initial value to this factor.
