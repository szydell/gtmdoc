PRIORITY

PRI[ORITY]=intlit

The integer literal specifies the base priority at which the resultant job executes.

For the JOB command to raise the PRIORITY of the resultant process above the PRIORITY of the parent process, the parent process must have the ALTPRI privilege.

The lowest priority is one (1). The highest priority is 31.

By default, the job has the current priority of the parent process.
