This chapter describes the M language commands implemented in GT.M. All of the commands starting with the letter Z are GT.M additions to the ANSI standard command set. The M standard specifies standard abbreviations for commands and rejects any non-standard abbreviation. For descriptions of the I/O commands OPEN, USE, READ, WRITE, and CLOSE, refer to the "Input/Output Processing" chapter in this manual.
