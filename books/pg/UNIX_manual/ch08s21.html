$ZALlocstor

$ZALLOCSTOR contains the number of bytes that are (sub) allocated (including overhead) by GT.M for various activities. It provides one view (see also “$ZREalstor” and “$ZUSedstor”) of the process memory utilization and can help identify storage related problems. GT.M does not permit $ZALLOCSTOR to be SET or NEWed.
