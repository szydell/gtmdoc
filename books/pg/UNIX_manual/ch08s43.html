$ZMOde

$ZMO[DE] contains a string value indicating the process execution mode.

The mode can be:

  • INTERACTIVE

  • OTHER

M routines cannot modify $ZMODE.

Example:

GTM>WRITE $ZMODE
INTERACTIVE

This displays the process mode.
