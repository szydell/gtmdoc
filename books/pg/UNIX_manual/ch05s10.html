Intrinsic Special Variables

M Intrinsic Special Variables start with a single dollar sign ($). GT.M provides such variables for program examination. In some cases, the Intrinsic Special Variables may be SET to modify the corresponding part of the environment. For information, see Chapter 8: “Intrinsic Special Variables”.
