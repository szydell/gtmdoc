$ZINInterrupt

$ZINI[NTERRUPT] evaluates to 1 (TRUE) when a process is executing code initiated by the interrupt mechanism, and otherwise 0 (FALSE).

GT.M does not permit the SET or NEW commands to modify $ZININTERRUPT.
